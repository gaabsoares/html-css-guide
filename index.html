HTML/CSS - References

HTML tags

Tag Name Description
<div> Div Defines a section in a document
<img> Image Defines an image
<Input> Input Defines an input control
<nav> Navigation Defines navigation links
<p> Paragraph Defines a paragraph

CSS attributes

Attribute Description
'color' Sets the text color for an element
'font-family' Defines the font family for text
'background-color' Defines the background color of an element
'margin' Specifies the margin space around an element
'padding' Defines the padding space inside an element
